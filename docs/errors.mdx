---
title: Errors
description: HTTP status codes and troubleshooting.
---

All errors return JSON with an `error` object:

```json
{
  "error": {
    "code": "insufficient_credits",
    "message": "Not enough credits for this generation"
  }
}
```

## Status codes

| Status | Meaning |
| --- | --- |
| `400` | Invalid request parameters |
| `401` | Missing or invalid API key |
| `402` | Insufficient credits |
| `404` | Generation not found |
| `409` | Idempotency conflict |
| `502` | Upstream generation failed |

## Common fixes

<AccordionGroup>
  <Accordion title="401 Unauthorized">
    Verify you are using a user API key (`ak_...`), not a Clerk secret key (`sk_live_...`). Create a new key in the dashboard if needed.
  </Accordion>
  <Accordion title="402 Payment Required">
    Buy credits on the [pricing page](https://seedanceapi.us/pricing). Check your balance with `GET /v1/credits`.
  </Accordion>
  <Accordion title="502 Upstream Error">
    Generation failed upstream. Retry the request, or try a different model or shorter duration.
  </Accordion>
</AccordionGroup>
